# <span style="color: #0048d8">**reset_mrc_header**</span>

### *summary*
---

Reset the pixel size in a MRC file header.

### *cli options*
---

**none**

### *interactive options*
---

**input_filename**

+ Filename of input image. Its header will be reset.

**new_pixel_size**

+ New pixel size, in Angstroms.
