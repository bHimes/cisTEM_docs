# <span style="color: #0048d8">**remove_inf_and_nan**</span>

### *summary*
---

Remove any infinite or NaN values from a volume.

### *cli options*
---

**none**

### *interactive options*
---

**input_filename**
  
+ Filename of input image.

**output_filename**
  
+ Filename of output image which should have no infs or nans.
